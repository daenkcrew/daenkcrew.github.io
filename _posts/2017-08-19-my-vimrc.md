A little backup for my `.vimrc` config file.

Other dotfiles can be found in [this repository](https://github.com/zackad/dotfiles).

```vim
" enable syntax highlighter
syntax on

" show line number
set number

" disable line wrap
set nowrap

" resize tab to 4 character
set expandtab ts=4 sw=4 ai
```
